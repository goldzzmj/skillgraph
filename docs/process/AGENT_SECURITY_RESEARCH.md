## 🔒 常见Agent Skills安全漏洞

### 1. 代码注入

**类型：**
- Python代码注入
- JavaScript代码注入
- 命令注入

**检测方法：**
- 静态分析：Semgrep, Bandit
- 动态分析：OWASP ZAP, Burp Suite
- 符号执行：SAGE, Angr

**影响：**
- 高危（Critical）
- 可以执行任意代码
- 数据泄漏
- 系统完全控制

---

### 2. 提示注入（Prompt Injection）

**类型：**
- 直接提示注入
- 间接提示注入
- 模型泄漏

**检测方法：**
- LLM扫描工具：Garak, LLMAP
- 红队工具：Rebuff
- 静态分析：自定义规则

**影响：**
- 高危（Critical）
- 绕过安全限制
- 数据泄漏
- 系统完全控制

---

### 3. 链注入（Chain Injection）

**类型：**
- LangChain链注入
- AutoGPT链注入
- 自定义链注入

**检测方法：**
- 框架特定检查：LangChain Security, AutoGPT Security
- 静态分析：自定义规则
- 动态分析：测试套件

**影响：**
- 中危（High）
- 链劫持
- 数据泄漏
- 行为异常

---

### 4. 权限提升（Privilege Escalation）

**类型：**
- 角色权限提升
- 资源访问提升
- API权限提升

**检测方法：**
- 静态分析：自定义规则
- 动态分析：权限测试
- 符号执行：路径分析

**影响：**
- 高危（High）
- 访问未授权资源
- 数据泄漏
- 系统损坏

---

### 5. 数据泄漏（Data Leakage）

**类型：**
- 敏感数据泄漏
- 凭据泄漏
- 配置信息泄漏

**检测方法：**
- 静态分析：数据流分析
- 动态分析：数据泄漏测试
- LLM扫描：模型泄漏检测

**影响：**
- 中危（Medium）
- 敏感信息暴露
- 合规风险
- 财务风险
